The output: Explained

Means

The means sheet will correctly formatted to be read into the software as a "means" sheet. It will contain the means of the collected data points for each sample for each element.

Full Statistical Report

This report shows the time value for each collected data point, and the data itself after drift correction for each element. It also displays the mean of those data points, the standard deviation of the corrected data and the percent standard error of the mean.
