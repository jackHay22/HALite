User Guide

Main Analysis Window

Getting Started

- To open a new project, go to File > Open... > New...

- To open an existing project, go to File > Open... > Saved...

View Options

- To truncate the displayed values in the calculated values panel (remove decimal places), select Window > Truncate model values .

Exporting Your Project

- With a project open, you have the option of exporting in a few different ways, go to File > Export... > and select your exporting method.

System Crashes

- In the event that the system crashes, you may be prompted to save a crash report file.

img:docs/images/critical_error.png:200:179

- For macOS users, selecting Save Log will write a crash report to UserHome/Library/Application Support/HALite. The text file saved to this directory will note the time and date when it was recorded (ex: crashreport_2018.12.01_13.53.58.txt)

- For Windows users, this crash report will be written to UserHome\AppData\Local\HALite

- Saving this crash report file will help the project developers locate and reproduce bugs in the system.

Drift Correction Window

Getting Started

- To open a new project, go to File > Drift Correction and select a file

- Proceed through the elements using the Previous Element and Next Element buttons or the Element dropdown

- Note: you will need to analyze all of the elements before exporting options are enabled

Exporting

- To export the dataset, go to File > Export and select a file

- To export the dataset and continue to the analysis window, go to File > Proceed to Analysis, select a file and then select the remaining csv files to open the analysis window
